# Logging Procedure Calculator v2.3 (Python)

Streamlit port of **LOGGING PROCEDURE CALCULATOR 2.3.html**.

## Run
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Features
- Multiple runs with per-run colour
- All-Runs combined view
- Condition background shading (Shut-in, Flowing, etc.)
- Y-axis compression for transit zones
- Long Station horizontal markers with dot-dash boundary lines
- Rotated X-axis pass boundary labels
- Export chart as PNG
- Save/Load project JSON (compatible with the HTML version)
